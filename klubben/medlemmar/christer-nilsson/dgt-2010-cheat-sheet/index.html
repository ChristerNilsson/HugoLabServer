Starta därefter med ⏯️ eller kontakta tävlingsledningen.  
De bör ha konfigurerat option 3 eller 18 före tävlingen.

Grund vänster 0 : 1 0 . 0 0
Grund höger 0 : 1 0 . 0 0
Bonus vänster 0 . 0 5
Bonus höger 0 . 0 5
